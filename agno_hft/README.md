# HFT Agno Agent 系統

智能化的高頻交易系統，通過Agno Agent框架實現自然語言驅動的交易流程。

## 系統架構

```
┌─────────────────────────────────────────────────────────────┐
│                    用戶自然語言輸入                           │
└─────────────────────┬───────────────────────────────────────┘
                     │
┌─────────────────────▼───────────────────────────────────────┐
│                 HFT Master Agent                            │
│              (主控制器 - 協調整個流程)                        │
└─────────────────────┬───────────────────────────────────────┘
                     │
        ┌────────────┼────────────┐
        │            │            │
┌───────▼──────┐ ┌───▼────┐ ┌────▼──────┐ ┌────────────┐
│ 訓練Agent     │ │評估Agent│ │風險Agent  │ │ 其他Agent   │
│ (模型訓練)    │ │(性能評估)│ │(風險控制) │ │ (可擴展)    │
└───────┬──────┘ └───┬────┘ └────┬──────┘ └────────────┘
        │            │            │
        └────────────┼────────────┘

┌─────────────────────▼───────────────────────────────────────┐

│                 Rust HFT Engine                            │
│              (核心交易引擎 - <50μs延遲)                      │
└─────────────────────────────────────────────────────────────┘
```

## 主要特性

### 🤖 智能Agent協調
- **HFT Master Agent**: 主控制器，負責理解用戶意圖並協調整個流程
- **Training Agent**: 專業的模型訓練專家
- **Evaluation Agent**: 模型性能評估專家
- **Risk Management Agent**: 風險控制專家

### 🚀 完整交易流程
1. **模型訓練**: 自動訓練LOB Transformer模型
2. **性能評估**: 全面評估模型表現
3. **乾跑測試**: 安全的模擬交易測試
4. **實盤部署**: 謹慎的實盤交易部署

### ⚡ 高性能架構
- **PyO3集成**: 零開銷的Rust-Python綁定
- **低延遲**: 目標推理延遲<50μs
- **內存優化**: 零拷貝數據共享
- **並行處理**: 多Agent並行協作

## 快速開始

### 1. 環境準備

```bash
# 安裝Python依賴
pip install -r requirements.txt

# 編譯Rust HFT系統
cd ../rust_hft
maturin develop --release

# 回到agno_hft目錄
cd ../agno_hft
```

### 2. 系統測試

```bash
# 運行集成測試
python test_integration.py
```

### 3. 啟動系統

```bash
# 啟動HFT Agent系統
python start_hft.py

# 或者直接運行
python main.py
```

## 系統配置

### 風險控制參數
- 最大倉位: 10%或1000/資金總額
- 止損比例: 2%
- 日損失限制: 5%
- 每小時最大交易次數: 100

### 模型評估標準
- 預測準確率 > 60%
- Sharpe比率 > 1.5
- 最大回撤 < 5%
- 推理延遲 < 50μs

## 安全特性

### 多層風險控制
1. **事前風險評估**: 交易前全面風險分析
2. **實時監控**: 持續監控交易表現
3. **緊急停止**: 一鍵停止所有交易
4. **資金保護**: 嚴格的資金管理規則

### 部署安全
- 必須通過完整的評估和測試流程
- 實盤交易需要用戶明確確認
- 異常情況自動暫停交易

## 故障排除

### 常見問題

1. **Rust模塊導入失敗**
   ```bash
   cd ../rust_hft
   maturin develop --release
   ```

2. **Agent初始化失敗**
   - 檢查agno框架是否正確安裝
   - 確認環境變量配置

3. **系統狀態異常**
   - 重啟系統
   - 運行測試腳本診斷

### 日誌檢查
系統會自動記錄詳細日誌，包括：
- Agent對話記錄
- 交易執行狀態
- 錯誤和異常信息
