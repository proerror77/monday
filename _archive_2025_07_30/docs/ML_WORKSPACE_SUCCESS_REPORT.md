## 2. ML Workspace 架構

### 2.1 核心組件
```
ml_workspace/
├── agno.toml                 # Agno工作區配置
├── components/              # 核心ML組件
│   ├── data_loaders.py      # ClickHouse數據加載器
│   ├── feature_engineering.py # TLOB特徵工程器
│   ├── model_trainer.py     # TLOB模型訓練器
│   └── model_evaluator.py   # 模型評估器
├── workflows/               # Agno工作流
│   ├── training_workflow.py # 主訓練工作流
│   └── simple_training_workflow.py
├── models/                  # 訓練好的模型 (36個)
│   ├── best_model_acc_0.8900_*.pt # 最佳模型 (89%準確率)
│   └── final_model_acc_0.8715_*.pt # 最終TorchScript模型
├── workspace/               # 基礎設施
│   └── infrastructure_client.py # ML基礎設施客戶端
└── test_full_training.py    # 完整訓練流程測試
```

### 2.2 技術架構
- **框架**: Agno Workflows v2 + PyTorch
- **數據源**: ClickHouse (真實市場數據)
- **模型**: TLOB Transformer (Top-Level Order Book)
- **特徵工程**: 27維混合特徵 (價格、成交量、技術指標、微觀結構)
- **通信**: Redis Pub/Sub + gRPC
