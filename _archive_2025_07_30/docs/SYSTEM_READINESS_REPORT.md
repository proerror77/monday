## 🚀 生產部署建議

### 1. 立即可執行的部署步驟

```bash
# 1. 啟動基礎設施
cd rust_hft && docker-compose up -d redis clickhouse

# 2. 啟動 Rust HFT 核心
cargo run --release --bin hft-core

# 3. 啟動 Ops Workspace (7x24監控)
cd ../ops_workspace && python3 -m agents.real_latency_guard

# 4. 配置 ML Workspace 每日訓練任務
cd ../ml_workspace && ag ws schedule training_workflow
```

### 2. 監控設置

**必要的監控面板**:
- Grafana Dashboard: `/rust_hft/config/grafana/`
- Prometheus 指標: `localhost:9090/metrics`
- Redis 監控: `redis-cli monitor`

**關鍵告警規則**:
```yaml
# 延遲告警
- alert: LatencySpike
  expr: hft_exec_latency_ms_p99 > 25e-3
  
# 系統告警  
- alert: SystemDown
  expr: up == 0
```

### 3. 風險控制配置

**自動停損設置**:
- 延遲超過 25μs → 自動降頻
- 延遲超過 50μs → 緊急停止
- 回撤超過 3% → 發送告警
- 回撤超過 5% → kill-switch
